# Report Structurer — 麦肯锡风格汇报重构器

一个 Claude Skill，将松散、逻辑感弱的汇报文字重构为 MECE、麦肯锡咨询风格的结构化汇报。

## 效果展示

**输入：**

> 今日对后台 3.6–3.8 期间用户反馈进行了集中处理，已对付费及非付费用户反馈进行整体排查，共处理约150条。当前仍有少量问题待技术侧进一步确认，用户反馈主要集中在订阅相关问题，包括部分用户权益未及时到账、下载乱码等情况。相关问题已同步记录并跟进处理中。明日计划：处理积压的邮箱用户反馈。

**输出：**

```
用户反馈处理日报

今日完成：
1. 3.6–3.8 版本用户反馈集中处理：完成付费 & 非付费用户反馈整体排查，共处理约 150 条
2. 高频问题定位：反馈主要集中在以下 3 类：
   - 订阅权益未到账：部分用户付费后权益未及时生效
   - 下载乱码：文件导出功能异常
3. 以上问题已记录并同步技术侧跟进

待处理：
1. 少量问题待技术侧进一步确认

明日计划：
1. 处理积压的邮箱渠道用户反馈

⚠️ 建议补充：待技术确认的问题具体数量和预计解决时间
```

## 适用场景

- 日报 / 周报 / 月报
- 会议纪要
- 产品功能更新
- 策略梳理
- 任意长段文字的逻辑整理

## 核心特性

- **MECE 分类**：自动按互不重叠的维度归类信息点
- **结构化输出**：数字编号 + 缩进 bullet，一目了然
- **纯文本格式**：可直接复制到飞书、钉钉、微信，不丢格式
- **场景自适配**：根据内容特征自动匹配日报/周报/月报/会议纪要等模板
- **信息补全提示**：自动识别逻辑断层，提示建议补充的信息

## 安装方式

1. 下载 `report-structurer.skill` 文件（见 [Releases](../../releases)）
2. 在 Claude 对话中上传该文件即可自动安装

或者手动安装：将 `report-structurer/SKILL.md` 放入你的 Claude Skills 目录。

## 使用方式

安装后，直接粘贴一段汇报文字给 Claude，Skill 会自动触发。你也可以说：

- "帮我整理一下这段汇报"
- "结构化一下"
- "理一下逻辑"
- "帮我写个周报"
